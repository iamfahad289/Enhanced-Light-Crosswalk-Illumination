Enhanced Light Crosswalk Illumination

Field evaluation of crosswalk illumination for improved nighttime pedestrian safety

Project Overview

This project evaluates an enhanced crosswalk illumination system designed to improve nighttime visibility of pedestrians, pavement markings, and crossing areas at stop and yield controlled locations.

Objectives

  • Improve pedestrian visibility during nighttime crossings
  • Quantify illumination performance using field measurements
  • Analyze driver response and safety related behavior

Data Collection

  • Continuous GNSS trajectory data
  • Road facing and driver facing video recordings
  • Vertical and horizontal illuminance measurements
  • Pedal based driver reaction logs

Outputs

  • Field deployment summaries
  • Figures and plots for technical presentations
  • Dataset documentation and analysis notes
